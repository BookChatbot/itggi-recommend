## 책 추천 챗봇 잇끼의 추천 알고리즘 구현 (Recommendation System)

### CB: Contents-Based

- 줄거리 기반의 추천 시스템
- 최근의 읽은 책을 기준으로 비슷한 내용의 책을 추천

### CF: Collaborative Filtering

- 최근접 이웃(Nearest-Neighbor) 기반 필터링
  - User-based
  - Item-based
- 잠재(Latent)요인 기반 필터링

### Hybrid

- CB + CF 모델의 결합
- LightFM 모델로 user feature와 book feature 기반으로 추천
